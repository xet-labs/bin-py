## bin/py

### Python Utilities

Python scripts for automation, monitoring, and system interaction.

| Script             | Description                                                                      |
| ------------------ | -------------------------------------------------------------------------------- |
| **sysmon.py**      | Gathers system stats like CPU/mem usage, outputs JSON or Prometheus metrics.     |
| **netcheck.py**    | Tests internet connectivity, pings servers, and prints latency and uptime stats. |
| **jsonfmt.py**     | Pretty-prints, validates, and reformats JSON files.                              |
| **redis-tools.py** | CLI wrapper for Redis operations like key scan, cache clear, and JSON get/set.   |
